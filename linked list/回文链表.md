# Problem Description
给你一个单链表的头节点 head，请你判断该链表是否为回文链表。如果是，返回 true；否则，返回 false。

回文序列是指正序（从左向右）和倒序（从右向左）读都相同的序列。

数据范围：

链表中节点的数目范围是 [1, 100000]

0 <= Node.val <= 9

给你一个单链表的头节点 head，请你判断该链表是否为回文链表。


# Key Points
注意要新建链表，不然会reverse修改原链表


# Code

## LC version

```python
class Solution:
    def isPalindrome(self, head: Optional[ListNode]) -> bool:
        # 回文：list = reverse(list)即可

        # 一定要复制链表，不然reverse会破坏原链表
        dummy = ListNode(-1)
        p = head
        q = dummy
        while p:
            q.next = ListNode(p.val)
            p = p.next
            q = q.next
            
        reversed_head = self.reverse(dummy.next)
        
        left = head
        right = reversed_head
        while left and right:
            if left.val != right.val:
                return False
            left = left.next
            right = right.next
            
        return True
        
    def reverse(self, head: Optional[ListNode]) -> Optional[ListNode]:
        if head is None or head.next is None:
            return head
        last = self.reverse(head.next)
        head.next.next = head
        head.next = None
        return last
```




# Complexity Analysis
- 时间复杂度：O(n)
- 空间复杂度：O(n)
