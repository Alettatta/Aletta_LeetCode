# Problem Description
给你单链表的头节点 head，请将其按 升序 排列并返回 排序后的链表。

数据范围：

链表中节点的数目范围是 [0, 5 * 10^4]

-10^5 <= Node.val <= 10^5

给你单链表的头节点 head，请将其按 升序 排列并返回 排序后的链表。


# Key Points
用递归思想，把链表从中间一分为二，分别排好左右两半，然后合成有序链表

把链表从中间一分为二，使用slow（一次一步）和fast（一次两步），这样fast到最后一个时候哦，slow在中点

slow从head开始，fast从head.next开始，不然只有2个节点的时候，无法递归下去，死循环


# Code

## LC version

```python
class Solution:
    def sortList(self, head: Optional[ListNode]) -> Optional[ListNode]:
        # 使用递归思想，把链表从中间一分为二，分别排好左右两半，然后合成有序链表
        # 把链表从中间一分为二，使用slow（一次一步）和fast（一次两步），这样fast到最后一个时候哦，slow在中点
        # slow从head开始，fast从head.next开始，不然只有2个节点的时候，无法递归下去，死循环

        # 特殊情况，没有节点或者只有一个结点
        if head is None or head.next is None:
            return head

        slow = head
        fast = head.next
        while fast is not None and fast.next is not None:
            slow = slow.next
            fast = fast.next.next
        mid = slow.next # 开始是右边部分了
        slow.next = None # 切断左右

        left = self.sortList(head)
        right = self.sortList(mid)

        return self.mergeList(left, right)

    def mergeList(self, l1: Optional[ListNode], l2: Optional[ListNode]) -> Optional[ListNode]:
        dummy = ListNode(-1)
        p1 = l1
        p2 = l2
        p = dummy
        while p1 is not None and p2 is not None:
            if p1.val < p2.val:
                p.next = p1
                p1 = p1.next 
            else:
                p.next = p2
                p2 = p2.next
            p = p.next
        if p1 is not None: # 假如l1只有[1]，p1就直接挪到了None；所以不是p1.next is None
            p.next = p1
        if p2 is not None:
            p.next = p2
        return dummy.next
```



# Complexity Analysis
- 时间复杂度：O(nlogn)，递归把链表对半分，递归树高度为 O(logn)；每一层所有合并操作的总代价是 O(n)，所以总复杂度是 O(nlogn)。
- 空间复杂度：O(logn)，递归调用栈的深度
