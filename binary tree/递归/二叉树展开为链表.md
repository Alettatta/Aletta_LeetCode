# Problem Description
给你二叉树的根结点 root，请你将它原地展开为一个单链表：

展开后的单链表应该同样使用 TreeNode，其中 right 子指针指向链表中下一个结点，而左子指针 left 始终为 null。

展开后的单链表应该与二叉树先序遍历顺序相同。


# Key Points
二叉树解题的思维模式分两类：

1、是否可以通过遍历一遍二叉树得到答案？如果可以，用一个 traverse 函数配合外部变量来实现，这叫「遍历」的思维模式。

2、是否可以定义一个递归函数，通过子问题（子树）的答案推导出原问题的答案？如果可以，写出这个递归函数的定义，并充分利用这个函数的返回值，这叫「分解问题」的思维模式。

无论使用哪种思维模式，你都需要思考：

（1）如果单独抽出一个二叉树节点，它需要做什么事情？

（2）需要在什么时候（前/中/后序位置）做？

（3）其他的节点不用你操心，递归函数会帮你在所有节点上执行相同的操作。

总结如果使用“遍历”的思维方式：

1. 这题能不能用「遍历」的思维模式解决？

2. 单独抽出一个节点，需要让它做什么？

3. 需要在什么时候做（前中后序位置）？

总结如果使用“分解”的思维方式：

1. 这题能不能用「递归」的思维模式解决？

2. 单独抽出一个节点，需要让它做什么？

3. 需要在什么时候做（前中后序位置）？

# Solution

1. 遍历的思维（可以使用）：先序遍历每一个节点——其中left指向Null，right指向下一个节点；什么时候做？先序位置，操作好了再遍历左右节点

代码如下：

```python
class Solution:
    def flatten(self, root: Optional[TreeNode]) -> None:        
        if root is None:
            return
            
        dummy = TreeNode(-1)
        self.p = dummy

        self.traverse(root)
            
    def traverse(self, root: Optional[TreeNode]):
        if root is None:
            return
        else:
            self.p.right = TreeNode(root.val)
            self.p = self.p.right
            self.traverse(root.left)
            self.traverse(root.right)
```

PS. 什么时候加“self.” 

① 在类的方法之间共享数据，p在同一个类的flatten和traverse之间共享

② 调用同类的其他方法，比如self.traverse

上述代码的问题：空间复杂度 O(n)，而 flatten 函数的签名返回类型为 void，也就是说题目希望我们在原地把二叉树拉平成链表。

2. 递归的思维（✅️）：因为left指向null而right指向下一个节点

对于每一个节点 x，可以先利用 flatten(x.left) 和 flatten(x.right) 将 x 的左右子树拉平；然后将整个左子树作为右子树，然后将原先的右子树接到当前右子树的末端。

# Code

## LC version

```python
class Solution:
    def flatten(self, root: Optional[TreeNode]) -> None:
        if root is None:
            return

        # 先展平左右子树
        self.flatten(root.left)
        self.flatten(root.right)

        # 把左子树接在右子树后面
        # step1:取左右子树
        left = root.left
        right = root.right
        # step2:右子树拿过来先接上
        root.left = None
        root.right = left
        # step3：原右子树接在现在右子树末端
        p = root
        while p.right is not None:
            p = p.right # 怎么取末端？取到最后一个位置，对应图上4 
        p.right = right # 接上
```

# Complexity Analysis
- 时间复杂度：O(n)
- 空间复杂度：O(1)
