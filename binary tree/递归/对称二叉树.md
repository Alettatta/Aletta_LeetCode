# Problem Description
给定一个二叉树的根节点 root，检查它是否轴对称（即沿中心镜像对称）。


# Key Points
递归本质：把规模较大的问题分解成规模较小的问题和，然后通过子问题的解得到原问题的解，我们可以称这种思维模式为「分解问题」。

如果你想用「分解问题」的思维模式来写递归算法，那么这个递归函数一定要有一个清晰的定义，说明这个函数参数的含义是什么，返回什么结果。


# Solution
递归：整棵树对称 -> 左右子树都对称，这就是isSymmetric函数作用

问题：如何判断一棵树对称？需要输入其左右子树，左右子树形状和值都一模一样，这就是check函数作用


# Code

## LC version

```python
class Solution:
    def isSymmetric(self, root: Optional[TreeNode]) -> bool:
        if root is None or (root.left is None and root.right is None):
            return True
        return self.check(root.left, root.right)

    def check(self, left: TreeNode, right: TreeNode) -> bool:
        if left is None or right is None:
            return left == right
        if left.val != right.val:
            return False
        return self.check(left.right, right.left) and self.check(left.left, right.right)
```



# Complexity Analysis
- 时间复杂度：O(n)
- 空间复杂度：O(n)
