# Problem Description

给定一个二叉树，找到该树中两个指定节点的最近公共祖先。

百度百科中最近公共祖先的定义为：「对于有根树 T 的两个节点 p、q，最近公共祖先表示为一个节点 x，满足 x 是 p、q 的祖先且 x 的深度尽可能大（一个节点也可以是它自己的祖先）。」

# Key Points
先不管最近公共祖先问题，我请你实现一个简单的算法：给你输入一棵没有重复元素的二叉树根节点 root 和一个目标值 val，请你写一个函数寻找树中值为 val 的节点。

因为我们希望时间复杂度尽可能低，所以最好不用遍历所有元素，比如在左子树找到了就没有必要去右子树找

```python
        # right找到了

        if right:
            return right

        return None
```

现在针对LCA问题，其实就是给了可能的两个值val1和val2，看能不能找到：

```python
def find(self, root: Optional[TreeNode], val1: int, val2: int) -> Optional[TreeNode]:
    if root is None:
        return None

    if root.val == val1 or root.val == val2:
        return root

    left = self.find(root.left, val1, val2)
    right = self.find(root.right, val1, val2)

    return left if left is not None else right
```

当然，这种写法会有重复遍历的问题，不过先不急着优化，最近公共祖先的一系列算法问题还就得基于这种写法展开。


# Solution
如果一个节点能够在它的左右子树中分别找到 p 和 q，则该节点为 LCA（Lowest Common Ancestor） 节点。

那不就是给定root，p和q的值，看能不能找到？



# Code

## LC version

```python
class Solution:
    def lowestCommonAncestor(self, root: 'TreeNode', p: 'TreeNode', q: 'TreeNode') -> 'TreeNode':
        return self.find(root, p.val, q.val)

    def find(self, root: Optional[TreeNode], val1: int, val2: int) -> Optional[TreeNode]:
        if root is None:
            return None
       
        if root.val == val1 or root.val == val2:
            return root

        left = self.find(root.left, val1, val2)
        right = self.find(root.right, val1, val2)

        if left is not None and right is not None:
            return root

        return left if left is not None else right
```



# Complexity Analysis
- 时间复杂度：O(n)
- 空间复杂度：O(1)
