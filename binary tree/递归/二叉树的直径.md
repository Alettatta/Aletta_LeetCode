# Problem Description
给定一棵二叉树的根节点 root，返回该二叉树的直径。

二叉树的直径是指树中任意两个节点之间最长路径的长度。这条路径可能经过也可能不经过根节点。

两节点之间路径的长度由它们之间边的数目表示。


# Solution
1. 直径 = 树中任意两个节点之间的最长路径长度，对于每个节点，找出"以该节点为最高点"的最长路径，取最大值
2. 对于任意节点 root，如果我们知道：leftMax = 左子树的最大深度（从 root.left 出发能走多远）rightMax = 右子树的最大深度（从 root.right 出发能走多远）那么以 root 为最高点的最长路径 = leftMax + rightMax
3. 现在出现一个矛盾：为了计算当前节点的直径，我需要左右子树的深度；为了计算父节点的直径，父节点需要知道我的深度；所以每个递归函数必须返回：当前节点的最大深度（从当前节点出发，到叶子节点的最长路径边数）；而直径作为一个全局统计量，用外部变量累计。
4. 递归函数：maxDepth(root):
    1. 如果 root 为空，深度为 0
    2. 递归计算左子树深度 leftMax
    3. 递归计算右子树深度 rightMax
    4. 用 leftMax + rightMax 更新全局最大直径
    5. 返回 1 + max(leftMax, rightMax) 给父节点使用
PS. 因为要先用子问题的结果（左右子树深度）才能计算当前节点的直径，所以必须后序遍历（先处理左右子树，再处理当前节点）。

示例：

```python
    1
   / \
  2   3
 / \
4   5
递归执行过程（从叶子开始）：
节点4：leftMax=0, rightMax=0 → 直径=0 → 返回深度1
节点5：leftMax=0, rightMax=0 → 直径=0 → 返回深度1
节点2：leftMax=1（来自4）, rightMax=1（来自5）→ 直径=2 → 返回深度2
节点3：leftMax=0, rightMax=0 → 直径=0 → 返回深度1
节点1：leftMax=2（来自2）, rightMax=1（来自3）→ 直径=3 → 返回深度3
最大直径 = max(0, 0, 2, 0, 3) = 3 ✅
路径是 4→2→1→3（3条边）
```

# Code

## LC version

```python
class Solution:
    def diameterOfBinaryTree(self, root: Optional[TreeNode]) -> int:
        if root is None:
            return 0
            
        # 左右子树的最大高度
        leftMax = self.maxDepth(root.left)
        rightMax = self.maxDepth(root.right)
        # root 的直径
        ans = leftMax + rightMax
        
        # 递归
        return max(ans, max(self.diameterOfBinaryTree(root.left),
                            self.diameterOfBinaryTree(root.right)))

                   
    def maxDepth(self, root: Optional[TreeNode]) -> int:
        if root is None:
            return 0
        else:
            leftMax = self.maxDepth(root.left)
            rightMax = self.maxDepth(root.right)
            return 1 + max(leftMax, rightMax)
```
- 时间复杂度：O(n^2)
- 空间复杂度：O(n)
  
更高效的做法：

```python
class Solution:
    def __init__(self):
        self.maxDiameter = 0

    def diameterOfBinaryTree(self, root: TreeNode) -> int:
        self.maxDepth(root)
        return self.maxDiameter

    def maxDepth(self, root: TreeNode) -> int:
        if root is None:
            return 0
        leftMax = self.maxDepth(root.left)
        rightMax = self.maxDepth(root.right)
        # 后序遍历位置顺便计算最大直径
        self.maxDiameter = max(self.maxDiameter, leftMax + rightMax)
        return 1 + max(leftMax, rightMax)
```

- 时间复杂度：O(n)
- 空间复杂度：O(n)
