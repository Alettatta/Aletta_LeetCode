# Problem Description
给定一个二叉树的根节点 root，想象自己站在它的右侧，按照从顶部到底部的顺序，返回从右侧所能看到的节点值。

换句话说，对二叉树的每一层，返回该层最右侧的那个节点的值。



# Solution
层序遍历返回每个[]的[-1]即可


# Code

## LC version

```python
class Solution:
    def rightSideView(self, root: Optional[TreeNode]) -> List[int]:
        ans = []
        res= []
        
        if root is None:
            return []

        q = deque()
        q.append(root)

        while q:
            sz = len(q)
            ceng = []
            for i in range(sz):
                cur = q.popleft()
                ceng.append(cur.val)
                if cur.left is not None:
                    q.append(cur.left)
                if cur.right is not None:
                    q.append(cur.right)
            ans.append(ceng)
                
        for ceng in ans:
            res.append(ceng[-1])

        return res
```
注意看函数要求输出list，那就不能写return而是return []，[]（有列表但没有元素）不是none（连列表都没有）！

# Complexity Analysis
- 时间复杂度：O(n)
- 空间复杂度：O(n)
