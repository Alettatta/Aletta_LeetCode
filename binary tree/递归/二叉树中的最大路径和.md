# Solution
可以使用递归方法，全局的最大路径和 -> max(每个节点的最大路径和) -> 所以需要用一全局变量记录每一个结点的情况

每个节点：root.val + max(0, left贡献) + max(0, right贡献)（经过父节点的路径 = 左贡献 + 父节点值 + 右贡献），递归函数的职责就是计算单边的贡献

后序方法，因为我们需要先知道左右情况才能知道root

## LC version

```python
class Solution:
    def __init__(self):
        self.ans = float('-inf')
        
    def maxPathSum(self, root: Optional[TreeNode]) -> int:
        if root is None:
            return 0
        # 递归函数的作用：计算子问题，也就是其中一边的最大值
        self.oneSideMax(root)
        # 经过父节点的路径 = 左贡献 + 父节点值 + 右贡献
        return self.ans

    def oneSideMax(self, root: Optional[TreeNode]) -> int:
        if root is None:
            return 0
        left_max_sum = max(0, self.oneSideMax(root.left))
        right_max_sum = max(0, self.oneSideMax(root.right))
        self.ans = max(self.ans, root.val + left_max_sum + right_max_sum)
        return root.val + max(left_max_sum, right_max_sum)
```



# Complexity Analysis
- 时间复杂度：O(n)
- 空间复杂度：O(1)
