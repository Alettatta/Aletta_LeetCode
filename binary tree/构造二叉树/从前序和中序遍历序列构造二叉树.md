# Problem Description
给定两个整数数组 preorder 和 inorder，其中 preorder 是二叉树的前序遍历，inorder 是同一棵树的中序遍历。请构造二叉树并返回其根节点。


# 铺垫
https://leetcode.cn/problems/maximum-binary-tree/description/

给定一个不重复的整数数组 nums 。 最大二叉树 可以用下面的算法从 nums 递归地构建:

创建一个根节点，其值为 nums 中的最大值。

递归地在最大值 左边 的 子数组前缀上 构建左子树。

递归地在最大值 右边 的 子数组后缀上 构建右子树。

返回 nums 构建的 最大二叉树 。

```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def constructMaximumBinaryTree(self, nums: List[int]) -> Optional[TreeNode]:
        # 可以用递归解决，首先构造root
        # 接下来每个节点：max左边的数组是root.left，右边的是root.right
        if len(nums) == 0:
            return 

        p = max(nums)
        index = nums.index(p)
        root = TreeNode(p)
        root.left = self.constructMaximumBinaryTree(nums[:index])
        root.right = self.constructMaximumBinaryTree(nums[index+1:])
        return root
```
上述解题两点要提：
1. nums可以直接查找索引，语法为index = nums.index(某值)
2. nums[start:stop:step],start 省略 → 默认从头开始(0),stop 省略 → 默认到结尾,step 省略 → 默认为1

   所以nums[:index]等价于 nums[0:index]，取从开头到index(不含index)
   
   nums[index+1:]等价于 nums[index+1:len(nums)]，取从index+1到结尾
   
   如果是nums[::index]，省略了start和stop，表示 index 是步长，在整个nums中取，这不对！

   

# Solution
一开始想法如下；

1. 前序遍历：root-L-R；中序遍历：L-root-R；根据preorder[0]可以定位root，那么inorder这个root前面的就是左子树，右边就是右子树
2. 左子树和右子树同样根据preorder和inorder的关系找到root然后构造左右子树

写到这里出现了一个问题：我想继续递归构造，如果调用原始函数buildTree做不下去，因为输入分别是前序遍历和中序遍历————

那么应该自己构造一个函数，希望输入是前序遍历数组 preorder[preStart..preEnd]、中序遍历数组 inorder[inStart..inEnd]，构造二叉树，返回该二叉树的根节点

## LC version

```python
class Solution:
    def buildTree(self, preorder: List[int], inorder: List[int]) -> Optional[TreeNode]:
        if len(preorder) == 0  and len(inorder) == 0:
            return
        return self.build(preorder, 0, len(preorder) - 1, 
                         inorder, 0 , len(inorder) - 1)
    
    def build(self, preorder: List[int], preStart: int, preEnd: int,
             inorder: List[int], inStart: int, inEnd: int) -> Optional[TreeNode]:

        # 递归终止条件(base case)——不加上的话，当子树为空时(比如某个节点没有左子树或右子树),对应的区间会变成无效区间
        if preEnd < preStart:
            return None
        
        rootVal = preorder[preStart] # 每个节点如果是根结点，那就是对应的前序遍历的第一个数字
        index = inorder.index(rootVal) # root对应中序数组的索引
        leftSide = index - inStart # 左边的就是左子树

        # 构造根结点
        root = TreeNode(rootVal)

        # 构造左子树，取出在前序和中序数组中的数组构造
        root.left = self.build(preorder, preStart+1, preStart+leftSide,
                              inorder, inStart, index-1)
        # 构造右子树，取出在前序和中序数组中的数组构造
        root.right = self.build(preorder, preStart+leftSide+1, preEnd,
                               inorder, index+1, inEnd)
        
        return root
```

另外，如果考虑到数组中有值重复，就不能直接用 nums.index() 的做法，可以建立一个数组元素和索引的哈希表：

```python
class Solution:
    valToIndex = dict()

    def buildTree(self, preorder, inorder):
        for i in range(len(inorder)):
            self.valToIndex[inorder[i]] = i
        return self.build(preorder, 0, len(preorder) - 1,
                          inorder, 0, len(inorder) - 1)

    def build(self, preorder, preStart, preEnd,
               inorder, inStart, inEnd):
        if preStart > preEnd:
            return None

        rootVal = preorder[preStart]
        index = self.valToIndex[rootVal]
        leftSize = index - inStart

        root = TreeNode(rootVal)
        root.left = self.build(preorder, preStart + 1, preStart + leftSize,
                               inorder, inStart, index - 1)
        root.right = self.build(preorder, preStart + leftSize + 1, preEnd,
                                inorder, index + 1, inEnd)

        return root
```
