# Problem Description
给你一个整数数组 nums，其中元素按 严格升序 排列，请你将其转换为一棵 高度平衡 的二叉搜索树。

高度平衡 二叉树是指每个节点的左右两个子树的高度差的绝对值不超过 1 的二叉树。


# Key Points
1. 对于 BST 的每一个节点 node，左子树节点的值都比 node 的值要小，右子树节点的值都比 node 的值大。
2. 对于 BST 的每一个节点 node，它的左侧子树和右侧子树都是 BST。
3. **BST 的中序遍历结果是有序的（升序）。**




# Code

## LC version

```python
class Solution:
    # 递归思想，因为BST按照中序遍历是有序的，所以可以先把中间的节点挑出来作为root
    # 然后构造root的左右子树，往下面迭代
    
    def sortedArrayToBST(self, nums: List[int]) -> Optional[TreeNode]:
        return self.build(nums, 0, len(nums) - 1) # 知道了完整数组，需要用它得到BST

    def build(self, nums, left, right):
        if left > right:
            return None

        mid = (left + right) // 2
        root = TreeNode(nums[mid])

        root.left = self.build(nums, left, mid-1)
        root.right = self.build(nums, mid+1, right)

        return root
```



# Complexity Analysis
- 时间复杂度：O(n)
- 空间复杂度：O(1)
