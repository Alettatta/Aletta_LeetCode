# Problem Description
你这个学期必须选修 numCourses 门课程，记为 0 到 numCourses - 1 。

在选修某些课程之前需要一些先修课程。 先修课程按数组 prerequisites 给出，其中 prerequisites[i] = [ai, bi] ，表示如果要学习课程 ai 则 必须 先学习课程  bi 。

例如，先修课程对 [0, 1] 表示：想要学习课程 0 ，你需要先完成课程 1 。
请你判断是否可能完成所有课程的学习？如果可以，返回 true ；否则，返回 false 。



# Solution
prerequisites[i] = [ai, bi] 即 bi -> ai，首先建立图

然后遍历图（DFS/BFS）都可以

如果没有环就是 True，反之 False



# Code

## LC version

### DFS版本
首先我们需要建图，可以使用邻接表和邻接矩阵，下面使用邻接表

```python
from typing import List
def buildGraph(numCourses: int, prerequisites: List[List[int]]) -> List[List[int]]:
    graph = [[] for _ in range(numCourses]
    for edge in prerequisites:
        start, end = edge[1], edge[0]
        graph[start].append(end)
    return graph
```

注意图中并不是所有节点都相连，所以要用一个 for 循环将所有节点都作为起点调用一次 DFS 搜索算法。

假设现在你以节点 2 为起点遍历所有可达的路径，最终发现没有环。假设另一个节点 5 有一条指向 2 的边，你再以 5 为起点遍历所有可达的路径时，肯定还会走到 2，那么请问，此时你是否还需要继续遍历 2 的所有可达路径呢？————如果我们发现一个节点之前被遍历过，就可以直接跳过，不用再重复遍历了。

另外，	onPath指的是当前这一条递归路径上正在访问的节点（进入节点时标记 True，递归返回后立刻撤销，作用是检测环），visited是历史上曾经被访问过的节点（一旦标记 True，永远不撤销，整个 canFinish 过程中只会变一次，作用是避免重复遍历，做剪枝优化）

```python
class Solution:
    def __init__(self):
        self.onPath = [] # 记录一次递归堆栈中节点，一个课可以是多门课的前置课程
        self.visited = []
        self.hasCycle = False

        
    def buildGraph(self, numCourses: int, prerequisites: List[List[int]]) -> List[List[int]]:
        graph = [[] for _ in range(numCourses)]
        for edge in prerequisites:
            start, end = edge[1], edge[0]
            graph[start].append(end)
        return graph

        
    def canFinish(self, numCourses: int, prerequisites: List[List[int]]) -> bool:
        # prerequisites[i] = [ai, bi] 看做 bi -> ai，根据pre数组画出图，图中没有环就true
        graph = self.buildGraph(numCourses, prerequisites)

        self.onPath = [False] * numCourses
        self.visited = [False] * numCourses

        for i in range(numCourses):
            self.traverse(graph, i)
        return not self.hasCycle


    # 遍历图中所有路径
    def traverse(self, graph: List[List[int]], s: int):
        if self.hasCycle:
            return # 找到环就不用遍历
            
        if self.onPath[s]: # s 已经在递归路径上，说明成环了
            self.hasCycle = True
            return 

        if self.visited[s]: # 不用再重复遍历已遍历过的节点
            return

        self.visited[s] = True
        self.onPath[s] = True
        for t in graph[s]:
            self.traverse(graph, t)
        self.onPath[s] = False # 这条路径结束了，回退，但是visited永远不更改
```


## ACM version

**ACM 模式的注意点：**

- 需要 import 完整的类（包括 sys、typing 等）
- 数据在标准输入流 stdin 中，全部是原始的文本字符串
- 必须用 print() 手动将结果写到标准输出流 stdout
- 需要写 while 或 for line in sys.stdin 循环处理，直到文件结束（EOF）

```python
import sys
from collections import deque
from typing import List




class Solution:
    def __init__(self):
        self.onPath = [] # 记录一次递归堆栈中节点，一个课可以是多门课的前置课程
        self.visited = []

    def buildGraph(self, numCourses: int, prerequisites: List[List[int]]) -> List[List[int]]:
        graph = [[] for _ in range(numCourses)]
        for edge in prerequisites:
            start, end = edge[1], edge[0]
            graph[start].append(end)
        return graph

        
    def canFinish(self, numCourses: int, prerequisites: List[List[int]]) -> bool:
        # prerequisites[i] = [ai, bi] 看做 bi -> ai，根据pre数组画出图，图中没有环就true
        graph = self.buildGraph(numCourses, prerequisites)

        self.onPath = [False] * numCourses

        for i in range(numCourses):
            self.traverse(graph, i)
        return not self.hasCycle


    # 遍历图中所有路径
    def traverse(self, graph: List[List[int]], s: int):
        if self.hasCycle:
            return # 找到环就不用遍历
            
        if self.onPath[s]: # s 已经在递归路径上，说明成环了
            self.hasCycle = True
            return 

        if self.visited[s]: # 不用再重复遍历已遍历过的节点

for line in sys.stdin:
    parts = line.split()
    if not parts:
        continue
    n, m = int(parts[0]), int(parts[1])
    prerequisites = []
    for _ in range(m):
        edge_line = sys.stdin.readline().split()
        a, b = int(edge_line[0]), int(edge_line[1])
        prerequisites.append([a, b])
    result = Solution().canFinish(n, prerequisites)

    print("true" if result else "false")
```
