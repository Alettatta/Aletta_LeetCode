# Problem Description
给定一个经过编码的字符串，返回它解码后的字符串。

编码规则为：k[encoded_string]，表示其中方括号内部的 encoded_string 正好重复 k 次。注意 k 保证为正整数。

你可以认为输入字符串总是有效的；输入字符串中没有额外的空格，且输入的方括号总是符合格式要求的。

此外，你可以认为原始数据不包含数字，所有的数字只表示重复的次数 k，例如不会出现像 3a 或 2[4] 的输入。

测试用例保证输出的长度不会超过 10^5。





# Code

## LC version

```python
class Solution:
    def decodeString(self, s: str) -> str:
        # 一个栈记录数字，一个栈记录字母；遇到[则加入字母栈
        # 遇到]的时候先出字母（直到遇到[）待定，然后出数字重复“数字”次；最后出字母栈中所有元素
        num = []
        char = []
        
        i = 0
        while i < len(s):
            c = s[i]
            if c.isalpha():
                char.append(c)
                i += 1
            elif c.isdigit():
                # 提取完整的数字
                repeat = ''
                while i < len(s) and s[i].isdigit():
                    repeat += s[i]
                    i += 1
                num.append(int(repeat))  # 存储完整的数字
            elif c == "[":
                char.append(c)
                i += 1
            elif c == ']':
                # 弹出字母
                temp = ''
                while char and char[-1] != '[':
                    temp = char.pop() + temp
                char.pop()  # 出 [
                
                # 弹出数字（完整的数字）
                repeat = num.pop()
                
                temp = temp * repeat
                
                # 压回去
                for ch in temp:
                    char.append(ch)
                i += 1
        
        return ''.join(char)
```

## ACM version

**ACM 模式的注意点：**

- 需要 import 完整的类（包括 sys、typing 等）
- 数据在标准输入流 stdin 中，全部是原始的文本字符串
- 必须用 print() 手动将结果写到标准输出流 stdout
- 需要写 while 或 for line in sys.stdin 循环处理，直到文件结束（EOF）
