# Problem Description
设计一个支持 push、pop、top 操作，并能在常数时间内检索到最小元素的栈。

实现 MinStack 类：

MinStack()：初始化堆栈对象。

void push(int val)：将元素 val 推入堆栈。

void pop()：删除堆栈顶部的元素。

int top()：获取堆栈顶部的元素。

int getMin()：获取堆栈中的最小元素。

push、pop、top 和 getMin 操作都必须以 O(1) 的时间复杂度运行。




# Code

## LC version

```python
class MinStack:
    # 从最小栈开始想，需要新建另一个栈minStack记录每个元素入栈的时候，当前栈中最小值
    
    def __init__(self):
        self.stack = []
        self.minStack = []

    def push(self, val: int) -> None:
        self.stack.append(val)
        # 一个元素入栈的时候如何求当前栈中最小值？类似马尔科夫，只要比前一个还小
        # 先检查 minStack 是否为空，避免访问 [-1] 时出现 IndexError
        if len(self.minStack) == 0 or val <= self.minStack[-1]:
            self.minStack.append(val)

    def pop(self) -> None:
        # 先确定minStack，如果要弹出的元素是minStack最小的，minStack也要变
        if self.stack[-1] == self.minStack[-1]:
            self.minStack.pop()
        self.stack.pop()

    def top(self) -> int:
        # 删除最上面的元素，其实是最后一个入栈的，也就是-1
        return self.stack[-1]

    def getMin(self) -> int:
        return self.minStack[-1]
```



# Complexity Analysis
- 时间复杂度：O(1)
- 空间复杂度：O(n)
