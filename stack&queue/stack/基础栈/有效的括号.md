# Problem Description
给定一个只包含字符 '('，')'，'{'，'}'，'['，']' 的字符串 s，判断字符串是否有效。

有效字符串需满足：

左括号必须用相同类型的右括号闭合。

左括号必须以正确的顺序闭合。

每个右括号都有一个对应的相同类型的左括号。



# Code

## LC version

```python
class Solution:
    def isValid(self, s: str) -> bool:
        # 建立一个栈，如果是([{就进栈，}])则对应的左括号出栈，不符合的话就是false
        
        stack = [] # 用列表实现栈，但是先进后出
        for element in s:
            if element in ["[","(","{"]:
                stack.append(element)
            else: # 说明是右括号，需要出栈匹配的左括号（出去：要保证栈里有元素）
                if stack and self.leftOf(element) == stack[-1]:
                    stack.pop()
                else:
                    return False
        return not stack # 如果只有一个（，还是没匹配上，要求此时stack为空才对，所以not stack

    def leftOf(self, element: str) -> str:
        if element == ")":
            return "("
        if element == "]":
            return "["
        if element == "}":
            return "{"
```

## ACM version

**ACM 模式的注意点：**

- 需要 import 完整的类（包括 sys、typing 等）
- 数据在标准输入流 stdin 中，全部是原始的文本字符串
- 必须用 print() 手动将结果写到标准输出流 stdout
- 需要写 while 或 for line in sys.stdin 循环处理，直到文件结束（EOF）

```python
import sys

class Solution:
    def isValid(self, s: str) -> bool:
        # 建立一个栈，如果是([{就进栈，}])则对应的左括号出栈，不符合的话就是false
        
        stack = [] # 用列表实现栈，但是先进后出
        for element in s:
            if element in ["[","(","{"]:
                stack.append(element)
            else: # 说明是右括号，需要出栈匹配的左括号（出去：要保证栈里有元素）
                if stack and self.leftOf(element) == stack[-1]:
                    stack.pop()
                else:
                    return False
        return not stack # 如果只有一个（，还是没匹配上，要求此时stack为空才对，所以not stack

    def leftOf(self, element: str) -> str:
        if element == ")":
            return "("
        if element == "]":
            return "["
        if element == "}":
            return "{"



for line in sys.stdin:
    # 不使用 strip()，避免在测试输入末尾保留换行；但要去掉行尾换行符
    s = line.rstrip('\n').rstrip('\r')

    result = Solution().isValid(s)

    print("true" if result else "false")
```


# Complexity Analysis
- 时间复杂度：O(n)
- 空间复杂度：O(n)
