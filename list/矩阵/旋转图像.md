# Problem Description
给定一个 _n  _× n 的二维矩阵 matrix 表示一个图像。请你将图像顺时针旋转 90 度。

你必须在**原地** 旋转图像，这意味着你需要直接修改输入的二维矩阵。请不要 使用另一个矩阵来旋转图像。


# Code

## LC version

```python
class Solution:
    def rotate(self, matrix: List[List[int]]) -> None:
        # 特殊规律：沿着左对角线翻转（行列标互换），然后反过来每一行即可
        
        n = len(matrix)
        
        # 沿着对角线镜像对称二维矩阵
        for i in range(n):
            for j in range(i, n): # j可遍历的数量越来越少
                matrix[i][j], matrix[j][i] = matrix[j][i], matrix[i][j]
        # 反过来每一行
        for row in matrix:
            self.reverse(row)

    def reverse(self, arr):
        i, j = 0, len(arr) - 1
        while j > i:
            arr[i], arr[j] = arr[j], arr[i]
            i += 1
            j -= 1
```

## ACM version

**ACM 模式的注意点：**

- 需要 import 完整的类（包括 sys、typing 等）
- 数据在标准输入流 stdin 中，全部是原始的文本字符串
- 必须用 print() 手动将结果写到标准输出流 stdout
- 需要写 while 或 for line in sys.stdin 循环处理，直到文件结束（EOF）

```python
import sys
from typing import List

class Solution:
    def rotate(self, matrix: List[List[int]]) -> None:
        # 特殊规律：沿着左对角线翻转（行列标互换），然后反过来每一行即可
        
        n = len(matrix)
        
        # 沿着对角线镜像对称二维矩阵
        for i in range(n):
            for j in range(i, n): # j可遍历的数量越来越少
                matrix[i][j], matrix[j][i] = matrix[j][i], matrix[i][j]
        # 反过来每一行
        for row in matrix:
            self.reverse(row)

    def reverse(self, arr):
        i, j = 0, len(arr) - 1
        while j > i:
            arr[i], arr[j] = arr[j], arr[i]
            i += 1
            j -= 1

n = 0
matrix = []
for line in sys.stdin:
    parts = line.strip().split()
    if not parts:
        continue
    if n == 0:
        n = int(parts[0])
        matrix = []
        continue
    row = [int(x) for x in parts]
    matrix.append(row)
    if len(matrix) == n:
        Solution().rotate(matrix)
        for r in matrix:
            print(' '.join(str(x) for x in r))
        n = 0
        matrix = []
```
