# Problem Description
给你一个整数数组 nums，请你找出所有满足 i != j、i != k 且 j != k，同时 nums[i] + nums[j] + nums[k] == 0 的不同三元组 [nums[i], nums[j], nums[k]]。

为保证答案唯一，输出需满足以下约束：

每个三元组内部的元素必须按照非递减顺序排列，即 [a, b, c] 满足 a <= b <= c。

所有三元组之间按字典序升序排列：先比较第一个元素，若相同再比较第二个元素，若仍相同再比较第三个元素。

答案中不能包含重复的三元组。

数据范围：

3 <= nums.length <= 3000

-10^5 <= nums[i] <= 10^5


# Key Points
把第一个数字固定，剩下两个数字target就知道了，核心还是两数之和，写一个嵌套的结果





# Code

## LC version

```python
class Solution:
    def threeSum(self, nums: List[int]) -> List[List[int]]:
        # 其实就是两数之和进阶版，对第一个数字进行穷举，后面还是使用两数之和做法，只是现在target==-第一个数
        # 因为这题要求输出不是索引，所以可以使用双指针sort
        
        nums.sort()
        ans = []

        index = 0
        while index < len(nums):
            tuples = self.twoSum(nums, index + 1, - nums[index]) # 满足条件的剩下两数
            for tuple in tuples:
                tuple.append(nums[index])
                ans.append(tuple)
            # 因为题目要求不能重复，所以要去掉nums[index]重复的情况
            while index < len(nums) - 1 and nums[index] == nums[index + 1]:
                index += 1
            index += 1
        return ans

    def twoSum(self, nums, start, target): # 已知从nums[start]开始看，给定target
        lo = start
        hi = len(nums) - 1
        res = []
        while lo < hi:
            sum = nums[lo] + nums[hi]
            left, right = nums[lo], nums[hi] # 记录lo和hi初始值
            if sum < target:
                while lo < hi and nums[lo] == left:
                    lo += 1
            elif sum > target:
                while lo < hi and nums[hi] == right:
                    hi -= 1
            else:
                res.append([left, right])
                while lo < hi and nums[lo] == left:
                    lo += 1
                while lo < hi and nums[hi] == right: 
                    hi -= 1
        return res
```

## ACM version

**ACM 模式的注意点：**

- 需要 import 完整的类（包括 sys、typing 等）
- 数据在标准输入流 stdin 中，全部是原始的文本字符串
- 必须用 print() 手动将结果写到标准输出流 stdout
- 需要写 while 或 for line in sys.stdin 循环处理，直到文件结束（EOF）

```python
import sys
from typing import List


class Solution:
    def threeSum(self, nums: List[int]) -> List[List[int]]:
        nums.sort()
        ans = []
        index = 0
        while index < len(nums):
            tuples = self.twoSum(nums, index + 1, -nums[index])
            for tuple in tuples:
                tuple.append(nums[index])
                ans.append(tuple)
            while index < len(nums) - 1 and nums[index] == nums[index + 1]:
                index += 1
            index += 1
        return ans
        
    def twoSum(self, nums, start, target):
        lo = start
        hi = len(nums) - 1
        res = []
        while lo < hi:
            sum = nums[lo] + nums[hi]
            left, right = nums[lo], nums[hi]
            if sum < target:
                while lo < hi and nums[lo] == left:
                    lo += 1
            elif sum > target:
                while lo < hi and nums[hi] == right:
                    hi -= 1
            else:
                res.append([left,right])
                while lo < hi and nums[lo] == left:
                    lo += 1
                while lo < hi and nums[hi] == right:
                    hi -= 1
        return res


data = sys.stdin.read().split()
idx = 0
while idx < len(data):
    n = int(data[idx])
    idx += 1
    nums = [int(data[idx + i]) for i in range(n)]
    idx += n
    result = Solution().threeSum(nums)
    triplets = sorted([sorted(t) for t in result])

    print(len(triplets))
    
    for t in triplets:
        print(f"{t[0]} {t[1]} {t[2]}")
```
