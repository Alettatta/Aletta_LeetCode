# Problem Description
给你一个整数数组 nums，将数组中的元素向右轮转 k 个位置，其中 k 是非负数。

返回轮转后的数组。



# Solution
最直接的想法是再开一个数组，把每个 nums[i] 放到 (i + k) % n 的位置，再拷贝回去，但这样需要 O(n) 的额外空间。有没有办法原地完成轮转呢？

观察一下旋转的结构：把 nums 切成前 n - k 个元素 A 和后 k 个元素 B 两段，向右轮转 k 步之后，结果就是 B 拼上 A，即 [B, A]。所以旋转的本质是让两段交换位置。

直接交换两段不好做，但「反转某一段」是可以原地完成的。关键性质在于：把 [A, B] 整体反转一次得到 [B', A']（' 表示反转），此时 B 已经被甩到了前面、A 被甩到了后面，只是各自的内部顺序反了。再把前 k 个和后 n - k 个分别反转回来，就得到了 [B, A]。

以 nums = [1, 2, 3, 4, 5, 6, 7]，k = 3 为例：整体反转得 [7, 6, 5, 4, 3, 2, 1]；反转前 3 个得 [5, 6, 7, 4, 3, 2, 1]；反转后 4 个得 [5, 6, 7, 1, 2, 3, 4]，正是答案。

# Code

## LC version

```python
class Solution:
    def rotate(self, nums: List[int], k: int) -> List[int]:
        # 尽量不要新开数组，而是原地修改数组
        n = len(nums)
        k = k % n # k可以大于n，先取模

        def reverse(left: int, right: int) -> None:
            while left < right:
                nums[left], nums[right] = nums[right], nums[left]
                left += 1
                right -= 1

        reverse(0, n - 1)
        reverse(0, k - 1)
        reverse(k, n - 1)
        return nums
```

## ACM version

**ACM 模式的注意点：**

- 需要 import 完整的类（包括 sys、typing 等）
- 数据在标准输入流 stdin 中，全部是原始的文本字符串
- 必须用 print() 手动将结果写到标准输出流 stdout
- 需要写 while 或 for line in sys.stdin 循环处理，直到文件结束（EOF）

```python
import sys
from typing import List

class Solution:
    def rotate(self, nums: List[int], k: int) -> List[int]:
        # 尽量不要新开数组，而是原地修改数组
        n = len(nums)
        k = k % n # k可以大于n，先取模

        def reverse(left: int, right: int) -> None:
            while left < right:
                nums[left], nums[right] = nums[right], nums[left]
                left += 1
                right -= 1

        reverse(0, n - 1)
        reverse(0, k - 1)
        reverse(k, n - 1)
        return nums

header = None
for line in sys.stdin:
    parts = line.strip().split()
    if not parts:
        continue
    if header is None:
        header = (int(parts[0]), int(parts[1]))
    else:
        n, k = header
        nums = list(map(int, parts))
        result = Solution().rotate(nums, k)
        print(' '.join(str(x) for x in result))
        header = None
```


# Complexity Analysis
- 时间复杂度：O(n)
- 空间复杂度：O(1)
