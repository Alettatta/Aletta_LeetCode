# Problem Description
给定一个数组 nums，编写一个函数将所有 0 移动到数组的末尾，同时保持非零元素的相对顺序。

请注意，必须在**不复制数组的情况下原地对数组进行操作**。


# Key Points
使用快慢指针，快指针fast在前面探路，维护数组/链表**从开始到slow是“有效”（满足题目要求）的**，注意终止条件（while）


# Solution
注意只能原地修改数组，不能重新建一个来存储


# Code

## LC version

```python
class Solution:
    def moveZeroes(self, nums: List[int]) -> None:
        # 快慢指针，fast在前面探路++1识别0，不是0的话slow移动到这个位置记录这个数字，是0的话记录0的个数，最后加上所有0

        fast = 0
        slow = 0
        
        while fast < len(nums):
            if nums[fast] != 0:
                nums[slow] = nums[fast]
                slow += 1
            fast += 1

        for i in range(slow,len(nums)):
            nums[i] = 0

        return nums
```

## ACM version

**ACM 模式的注意点：**

- 需要 import 完整的类（包括 sys、typing 等）
- 数据在标准输入流 stdin 中，全部是原始的文本字符串
- 必须用 print() 手动将结果写到标准输出流 stdout
- 需要写 while 或 for line in sys.stdin 循环处理，直到文件结束（EOF）

```python
# 下面这段代码适用于输入list，输出list元素字符串的问题
from typing import List
import sys

class Solution:
    def moveZeroes(self, nums: List[int]) -> None: # 因为题目说不用输出，所以是none
        fast = 0
        slow = 0
        while fast < len(nums):
            if nums[fast] != 0:
                nums[slow] = nums[fast]
                slow += 1
            fast += 1
        for i in range(slow, len(nums)):
            nums[i] = 0

data = sys.stdin.read().split() # 只是分隔开，还是字符串类型，['5', '0', '1', '0', '3', '12']
index = 0
while index < len(data):
    # 第一个数字表示数组长度
    n = int(data[index]) # 本来是字符串，需要转换为int
    index += 1
    # 后面是数组
    nums = [int(x) for x in data[index:index + n]] # 一口气写完整个数组，不用遍历
    index += n
    
    Solution().moveZeroes(nums)
        
    # 输出还是要求间隔开的字符
    print(' '.join(str(x) for x in nums))
```


# Complexity Analysis
- 时间复杂度：O(n)
- 空间复杂度：O(1)
