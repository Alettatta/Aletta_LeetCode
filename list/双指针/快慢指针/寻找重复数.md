# Problem Description
给你一个包含 n + 1 个整数的数组 nums，其中每个整数都在 [1, n] 范围内（包含 1 和 n）。可知数组中只有一个重复的整数，请找出这个重复的数。

注意：该重复的整数可能在 nums 中出现两次或多次。

数据范围：

1 <= n <= 10^5

nums.length == n + 1

1 <= nums[i] <= n

除了恰好一个整数会出现两次或多次外，其余整数均只出现一次。


# Key Points
重点是所有数字都在[1,n]之间，可以看成是一个单链表
```python
索引:  0 -> 1 -> 3 -> 2 -> 4
值:    1 -> 3 -> 2 -> 4 -> 2
                 ↑_________↓
```
本质是floyd环、龟兔赛跑找环的入口：第一阶段快慢指针相遇确定有环，第二阶段fast不变slow回到0找环入口索引


# Solution
```python
具体实例演示
nums = [1, 3, 4, 2, 2]

第一阶段：
初始：slow=0, fast=0
第1步：fast=nums[nums[0]]=nums[1]=3, slow=nums[0]=1
第2步：fast=nums[nums[3]]=nums[2]=4, slow=nums[1]=3
第3步：fast=nums[nums[4]]=nums[2]=4, slow=nums[3]=2
第4步：fast=nums[nums[4]]=nums[2]=4, slow=nums[2]=4
相遇在索引4！

第二阶段：
slow=0, fast=4
第1步：fast=nums[4]=2, slow=nums[0]=1
第2步：fast=nums[2]=4, slow=nums[1]=3
第3步：fast=nums[4]=2, slow=nums[3]=2
相遇在索引2！
答案就是 nums[2] = 2，重复数字找到了！
```


# Code

## LC version

```python
class Solution:
    def findDuplicate(self, nums: List[int]) -> int:
        # 不仅仅是找重复数字那么简单，关键条件是元素在[1,n]，那么可以认为nums的值和下标有对应关系
        # 类比为单链表val和next，让数组索引作为val（其实不重要），next指向数组值为nums[i]的节点（重要）
        # 重点就是按照数组的值->单链表的索引连起来（每个数字放到和它索引一样的地方），可以得到：
        # ListNode(0) -> ListNode(1) -> ListNode(3) -> ListNode(2) -> ListNode(4)
                                                #↑               |
                                                #└───────────────┘
                                              #（2 重复，成为环的入口）
        # 只需要找环入口即可————单链表环检测问题

        # 为了说明确实有环，快慢指针会相遇，找到相遇点的索引，用fast记录
        fast = slow = 0
        while True:
            fast = nums[nums[fast]]
            slow = nums[slow]
            if fast == slow:
                break

        # 找到环入口的数字
        slow = 0
        while slow != fast:
            fast = nums[fast]
            slow = nums[slow]
        return slow
```

## ACM version

**ACM 模式的注意点：**

- 需要 import 完整的类（包括 sys、typing 等）
- 数据在标准输入流 stdin 中，全部是原始的文本字符串
- 必须用 print() 手动将结果写到标准输出流 stdout
- 需要写 while 或 for line in sys.stdin 循环处理，直到文件结束（EOF）

```python
import sys
from typing import List

class Solution:
    def findDuplicate(self, nums:List[int]) -> None:
        slow = fast = 0
        while True:
            fast = nums[nums[fast]]
            slow = nums[slow]
            if fast == slow:
                break
                
        slow = 0
        while slow != fast:
            fast = nums[fast]
            slow = nums[slow]
        return slow

data = sys.stdin.read().split()
index = 0
while index < len(data):
    n = int(data[index])
    index += 1
    nums = [int(x) for x in data[index:index+n]]
    index += n
    print(Solution().findDuplicate(nums))
```


# Complexity Analysis
- 时间复杂度：O(n)
- 空间复杂度：O(1)
