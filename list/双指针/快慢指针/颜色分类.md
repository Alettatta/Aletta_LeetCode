# Problem Description
给你一个包含 n 个元素的数组 nums，数组中元素的颜色为红色、白色或蓝色。请你原地对它们进行排序，使得相同颜色的元素相邻，并按红色、白色、蓝色的顺序排列。

我们使用整数 0、1 和 2 分别表示红色、白色和蓝色。

必须在不使用库内置 sort 函数的情况下解决这个问题。

**进阶：你能想出一个仅使用常数空间的一趟扫描算法吗？**

数据范围：

n == nums.length

1 <= n <= 300

nums[i] 为 0、1 或 2


# Key Points
快慢指针核心：慢指针左侧是符合要求的部分，右边是待定的


# Solution
法1：两次移动0

法2：三个指针，两个慢指针两侧是符合要求的


# Code

## LC version（1）
```python
class Solution:
    def sortColors(self, nums: List[int]) -> None:
        # 快慢指针的核心：慢指针左边是符合要求的内容，右边是待定区
        # 这题有三个部分，可以第一次处理1放到末尾，第二次处理2
        fast = 0
        slow = 0
        while fast < len(nums):
            if nums[fast] != 1:
                nums[slow] = nums[fast]
                slow += 1
            fast += 1
        for i in range(slow, len(nums)):
            nums[i] = 1
        fast = 0
        slow = 0
        while fast < len(nums):
            if nums[fast] != 2:
                nums[slow] = nums[fast]
                slow += 1
            fast += 1
        for i in range(slow, len(nums)):
            nums[i] = 2
```


## LC version（2）
```python
class Solution:
    def sortColors(self, nums: List[int]) -> None:
        # 如果想只遍历一次数组就解决问题，根据“慢指针左侧是搞定了的”原理，可以维护px和py2个慢指针
        # 其中nums[0,px)是0，nums(py,len(nums)-1]是2，中间留一个快指针p保证为1
        # 三者数量关系（遍历结束）：px<=p<=py
        
        px = 0 # 保证最开始区间里没数
        py = len(nums) - 1 # 保证最开始区间里没数
        p = 0

        while p<=py:
            if nums[p] == 2:
                nums[py], nums[p] = nums[p], nums[py]
                py -= 1 # 表示这个位置的数字搞定了，但是p不变因为还要检验
            elif nums[p] == 0:
                nums[px], nums[p] = nums[p], nums[px]
                px += 1
            else:
                p += 1

            if p < px:
                p = px  
```

## ACM version

**ACM 模式的注意点：**

- 需要 import 完整的类（包括 sys、typing 等）
- 数据在标准输入流 stdin 中，全部是原始的文本字符串
- 必须用 print() 手动将结果写到标准输出流 stdout
- 需要写 while 或 for line in sys.stdin 循环处理，直到文件结束（EOF）

```python
import sys 
from typing import List

class Solution:
    def sortColors(self, nums:List[int]) -> None:
        p = 0
        px = 0
        py = len(nums) - 1
        while p <= py:
            if nums[p] == 0:
                nums[px], nums[p] = nums[p], nums[px]
                px += 1
            elif nums[p] == 2:
                nums[py], nums[p] = nums[p], nums[py]
                py -= 1
            else:
                p += 1
            if p < px:
                p = px

data = sys.stdin.read().split()
index = 0
while index < len(data):
    n = int(data[index])
    index += 1
    nums = [int(x) for x in data[index:index+n]]
    index += n
    Solution().sortColors(nums)
    print(" ".join(str(x) for x in nums))
```


# Complexity Analysis
- 时间复杂度：O(n)
- 空间复杂度：O(1)
