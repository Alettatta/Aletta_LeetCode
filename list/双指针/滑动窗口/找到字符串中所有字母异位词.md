# Problem
https://labuladong.online/zh/problem/leetcode/find-all-anagrams-in-a-string/description/


# Problem Description
给定两个字符串 s 和 p，找到 s 中所有 p 的 异位词 的子串，返回这些子串的起始索引。不考虑答案输出的顺序。


# Key Points
滑动窗口主要处理子串/子数组问题，三连问：
1. 什么时候扩大窗口（右移right）？窗口扩大时，要更新什么？
2. 什么时候缩小窗口（右移left）？窗口缩小时，要更新什么？
3. 什么时候更新结果？


# Solution
注意p给定说明window大小恒定，与经典模板的区分点在于满足结果的条件不一样


# Code

## LC version

```python
class Solution:
    def findAnagrams(self, s: str, p: str) -> List[int]:
        # 滑动窗口主要处理子串/子数组问题，三连问：
        # 1. 什么时候扩大窗口（右移right）？（子串长度）窗口扩大时，要更新什么？（window、right、valid、res）
        # 2. 什么时候缩小窗口（右移left）？（不是window和need的关系了，因为这题p是固定的，所以s里面子串长度固定，一旦大了就要缩小；这个时候再去判断valid）窗口缩小时，要更新什么？
        # 3. 什么时候更新结果？
        need, window = {}, {}
        for char in p:
            need[char] = need.get(char,0) + 1

        left, right = 0, 0
        valid = 0
        ans = []

        # 扩大窗口
        while right < len(s):
            char = s[right]
            right += 1
            if char in need:
                window[char] = window.get(char, 0) + 1
                if window[char] == need[char]:
                    valid += 1

            while right - left >= len(p):
                if valid == len(need):
                    ans.append(left)
                # 缩小窗口
                d = s[left]
                left += 1
                if d in need:
                    if need[d] == window[d]:
                        valid -= 1
                    window[d] -= 1
        
        return ans
```

## ACM version

**ACM 模式的注意点：**

- 需要 import 完整的类（包括 sys、typing 等）
- 数据在标准输入流 stdin 中，全部是原始的文本字符串
- 必须用 print() 手动将结果写到标准输出流 stdout
- 需要写 while 或 for line in sys.stdin 循环处理，直到文件结束（EOF）

```python
import sys
from typing import List

class Solution:
    def findAnagrams(self, s: str, p: str) -> List[int]:
        # 滑动窗口主要处理子串/子数组问题，三连问：
        # 1. 什么时候扩大窗口（右移right）？（子串长度）窗口扩大时，要更新什么？（window、right、valid、res）
        # 2. 什么时候缩小窗口（右移left）？（不是window和need的关系了，因为这题p是固定的，所以s里面子串长度固定，一旦大了就要缩小；这个时候再去判断valid）窗口缩小时，要更新什么？
        # 3. 什么时候更新结果？
        need, window = {}, {}
        for char in p:
            need[char] = need.get(char,0) + 1

        left, right = 0, 0
        valid = 0
        ans = []

        # 扩大窗口
        while right < len(s):
            char = s[right]
            right += 1
            if char in need:
                window[char] = window.get(char, 0) + 1
                if window[char] == need[char]:
                    valid += 1

            while right - left >= len(p):
                if valid == len(need):
                    ans.append(left)
                # 缩小窗口
                d = s[left]
                left += 1
                if d in need:
                    if need[d] == window[d]:
                        valid -= 1
                    window[d] -= 1
                    
        return ans

for line in sys.stdin:
    s = line.rstrip('\n').rstrip('\r')
    p = sys.stdin.readline().rstrip('\n').rstrip('\r')
    result = sorted(Solution().findAnagrams(s, p))
    print(' '.join(str(x) for x in result))
```
