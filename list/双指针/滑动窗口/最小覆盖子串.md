# Problem Description
给定两个字符串 s 和 t，长度分别是 m 和 n，返回 s 中的最短窗口子串，使得该子串包含 t 中的每一个字符（包括重复字符）。如果没有这样的子串，返回空字符串 ""。

测试用例保证答案唯一。


# Key Points
滑动窗口主要处理子串/子数组问题，三连问：
1. 什么时候扩大窗口（右移right）？窗口扩大时，要更新什么？
2. 什么时候缩小窗口（右移left）？窗口缩小时，要更新什么？
3. 什么时候更新结果？

# Code

## LC version

```python
class Solution:
    def minWindow(self, s: str, t: str) -> str:
        # 根据t建立need字典知道要求是什么，移动时候的记作window
        # 滑动窗口主要处理子串/子数组问题，三连问：
        # 1. 什么时候扩大窗口（右移right）？（不满足need）窗口扩大时，要更新什么？（window、valid）
        # 2. 什么时候缩小窗口（右移left）？（window>need）窗口缩小时，要更新什么？（window、valid、子串和长度）
        # 3. 什么时候更新结果？（window<need）

        need, window = {}, {}
        for char in t:
            need[char] = need.get(char,0) + 1 # 如果char不存在于need中就默认值0

        left = 0
        right = 0 
        valid = 0 # window中满足need键（char）的个数
        # 记录最小覆盖子串
        start = 0 
        length = float('inf')

        # 扩大窗口
        while right < len(s):
            char = s[right]
            right += 1
            if char in need:
                window[char] = window.get(char,0) + 1
                if window[char] == need[char]:
                    valid += 1

            # 缩小窗口
            while valid == len(need):
                if right - left < length:
                    start = left
                    length = right - left # 还在这个大窗口里面的缩小，没有重开left
                d = s[left] # 移出多余的字符
                left += 1
                if d in need:
                    if window[d] == need[d]:
                        valid -= 1
                    window[d] -= 1

        return '' if length == float('inf') else s[start: start+length]
```

## ACM version

**ACM 模式的注意点：**

- 需要 import 完整的类（包括 sys、typing 等）
- 数据在标准输入流 stdin 中，全部是原始的文本字符串
- 必须用 print() 手动将结果写到标准输出流 stdout
- 需要写 while 或 for line in sys.stdin 循环处理，直到文件结束（EOF）

```python
import sys
from collections import defaultdict


class Solution:
    def minWindow(self, s: str, t: str) -> str:
        need, window = {}, {}
        for char in t:
            need[char] = need.get(char,0) + 1

        left, right = 0, 0
        valid = 0
        start = 0
        length = float('inf')

        while right < len(s):
            char = s[right]
            right += 1
            if char in need:
                window[char] = window.get(char,0) + 1
                if window[char] == need[char]:
                    valid += 1

            while valid == len(need):
                # 要先更新子串再移出
                if right - left < length:
                    start = left
                    length = right - left
                d = s[left]
                left += 1
                if d in need:
                    if window[d] == need[d]:
                        valid -= 1
                    window[d] -= 1

        return '' if length == float('inf') else s[start:start+length]

        
for line in sys.stdin: # 因为都要保留字符串所以一行一行看
    # 注意：不使用 strip()，保留可能为空的字符串；只去掉行尾换行符
    s = line.rstrip('\n').rstrip('\r') # 处理第一行s
    t = sys.stdin.readline().rstrip('\n').rstrip('\r')
    result = Solution().minWindow(s, t)
    print(result)
```
