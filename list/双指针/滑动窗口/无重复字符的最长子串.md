# Problem Description
给定一个字符串 s，请你找出其中不含有重复字符的 最长子串 的长度。


# Key Points
滑动窗口主要处理子串/子数组问题，三连问：
1. 什么时候扩大窗口（右移right）？窗口扩大时，要更新什么？
2. 什么时候缩小窗口（右移left）？窗口缩小时，要更新什么？
3. 什么时候更新结果？


# Code

## LC version

```python
class Solution:
    def lengthOfLongestSubstring(self, s: str) -> int:
        # 扩大窗口：没有重复字符（window里面没有char的值>1），更新window、right
        # 缩小窗口：有重复字符（window里面有char的值=2），更新window、left

        ans = 0
        left, right = 0, 0
        window = {}
        
        while right < len(s):
            char = s[right]
            window[char] = window.get(char,0) + 1
            right += 1
            while window[char] > 1:
                d = s[left] # 因为尽管是右边的字符char不符合要求，我们只能按照滑动窗口改变左边的字符
                left += 1
                window[d] = window.get(d,0) - 1
            ans = max(ans, right - left)

        return ans
```

## ACM version

**ACM 模式的注意点：**

- 需要 import 完整的类（包括 sys、typing 等）
- 数据在标准输入流 stdin 中，全部是原始的文本字符串
- 必须用 print() 手动将结果写到标准输出流 stdout
- 需要写 while 或 for line in sys.stdin 循环处理，直到文件结束（EOF）

```python
import sys


class Solution:
    def lengthOfLongestSubstring(self, s: str) -> int:
        ans = 0
        left, right = 0, 0
        window = {}
        
        while right < len(s):
            char = s[right]
            window[char] = window.get(char,0) + 1
            right += 1
            while window[char] > 1:
                d = s[left] # 因为尽管是右边的字符char不符合要求，我们只能按照滑动窗口改变左边的字符
                left += 1
                window[d] = window.get(d,0) - 1
            ans = max(ans, right - left)

        return ans


for line in sys.stdin:
    # 不使用 strip()，保留可能为空的字符串；只去掉行尾换行符
    s = line.rstrip('\n').rstrip('\r')
    result = Solution().lengthOfLongestSubstring(s)
    print(result)
```


# Complexity Analysis
- 时间复杂度：O(n)
- 空间复杂度：O(n)
