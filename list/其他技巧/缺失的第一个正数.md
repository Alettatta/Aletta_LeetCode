# Problem Description
给你一个未排序的整数数组 nums，请你找出其中没有出现的最小的正整数。

请你实现时间复杂度为 O(n) 并且只使用常数级别额外空间（O(1)）的解决方案。


# Solution
n = len(nums)，最理想的情况是恰好装着 1, 2, ..., n，此时答案为 n+1

只要中间缺了任何一个，答案就落在 [1, n] 之间，所以答案一定在 [1, n+1] 这个区间内

这就意味着我们只关心 1 到 n 这 n 个数有没有出现过，而它们正好和数组下标 0 到 n-1 一一对应；可以把数组本身当哈希表：把数字 x 映射到下标 x-1，用这个位置上值的正负来标记 x 是否出现过

不过原数组里可能包含负数、零、大于 n 的数；先做一轮预处理，把所有无效数字统一替换成哨兵值 n+1

```python
class Solution:
    def firstMissingPositive(self, nums: List[int]) -> int:
        # 最小正整数是 1
        # 数组连续，且从<=1的数开始就连续，找最大值+1
        # 不管数组是否连续，只要是min>=2的一个区间，答案是 1
        # 数组不连续，看不连续在哪里，找缺失值
        # 问题：如何判断数组是否连续&不连续区间是？但是这样做可以不保证时间复杂度O(n)空间O(1

        # n = len(nums)，最理想的情况是恰好装着 1, 2, ..., n，此时答案为 n+1
        # 只要中间缺了任何一个，答案就落在 [1, n] 之间，所以答案一定在 [1, n+1] 这个区间内
        # 这就意味着我们只关心 1 到 n 这 n 个数有没有出现过，而它们正好和数组下标 0 到 n-1 一一对应
        # 可以把数组本身当哈希表：把数字 x 映射到下标 x-1，用这个位置上值的正负来标记 x 是否出现过
        # 不过原数组里可能包含负数、零、大于 n 的数，
        # 先做一轮预处理，把所有无效数字统一替换成哨兵值 n+1

        n = len(nums)
        # 把无效值（<=0或>n）的替换为哨兵n+1
        for i in range(n):
            if nums[i] <= 0 or nums[i] > n:
                nums[i] = n + 1

        # 用符号位标记，nums[x-1] 取负表示 x 出现过
        for i in range(n):
            x = abs(nums[i]) # 取绝对值是因为这个位置可能已被前面的迭代标记为负数
            if x <= n:
                nums[x - 1] = -abs(nums[x - 1])

        # 第一个正数位置 i 说明 i+1 缺失
        for i in range(n):
            if nums[i] > 0:
                return i + 1

        return n + 1 # 1..n 全都出现了，答案是 n+1
```

根据代码的思路讲解：

1. 预处理——把"无用的数"统一处理掉，因为我们只关心 1~n,所以把所有 <=0 或 >n 的数字,都替换成 n+1 这个"占位符"，例如 nums = [3, 4, -1, 1],n=4:结果:[3, 4, 5, 1]

2. 核心技巧——用"下标"当哈希表,用"正负号"当标记

数字 x(在 1~n 范围内)对应的下标是 x-1。如果 x 出现在数组里,就把下标 x-1 位置上的数变成负数,当作"打卡"标记。

继续上面的例子:nums = [3, 4, 5, 1],n=4

i=0: nums[0]=3,x=3,3<=4,把下标 3-1=2 的位置变负:nums[2] = -5 → 数组变成 [3, 4, -5, 1]
i=1: nums[1]=4,x=4,4<=4,把下标 4-1=3 的位置变负:nums[3] = -1 → 数组变成 [3, 4, -5, -1]
i=2: nums[2]=-5,取绝对值 x=5,5>4(超出范围),跳过
i=3: nums[3]=-1,取绝对值 x=1,1<=4,把下标 1-1=0 的位置变负:nums[0] = -3 → 数组变成 [-3, 4, -5, -1]

最终数组:[-3, 4, -5, -1]

3. 找第一个"没被标记"的位置（第一个还是正数的位置即可）

for line in sys.stdin:

    for token in parts:
