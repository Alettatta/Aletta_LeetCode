# Problem Description
给定一个大小为 n 的数组 nums，返回其中的 多数元素。多数元素是指在数组中出现次数 大于 ⌊n/2⌋ 的元素。

题目数据保证数组中的多数元素一定存在。




# Solution
如果没有众数，那么这堆电子一定电中性（count=0），众数来了就+1，反之-1，最后count肯定是正，此时指针一定指众数


# Code

## LC version

```python
class Solution:
    def majorityElement(self, nums: List[int]) -> int:
        # 本质就是在nums中找一个众数
        target = 0 # 目标数字
        count = 0 # 计数器（类比一群带电粒子：数量相等的初始状态就呈电中性）
        for i in range(len(nums)):
            if count == 0:
                target = nums[i] # 当计数器为 0 时，假设 nums[i] 就是众数
                count = 1
            elif nums[i] == target:
                count += 1 # 如果是target的话，类似正电子
            else:
                count -= 1 # 反之带负电
        return target # 因为存在众数，count必然>0，此时target就是众数
```

## ACM version

**ACM 模式的注意点：**

- 需要 import 完整的类（包括 sys、typing 等）
- 数据在标准输入流 stdin 中，全部是原始的文本字符串
- 必须用 print() 手动将结果写到标准输出流 stdout
- 需要写 while 或 for line in sys.stdin 循环处理，直到文件结束（EOF）

```python
import sys
from typing import List

class Solution:
    def majorityElement(self, nums: List[int]) -> int:
        # 本质就是在nums中找一个众数
        target = 0 # 目标数字
        count = 0 # 计数器（类比一群带电粒子：数量相等的初始状态就呈电中性）
        for i in range(len(nums)):
            if count == 0:
                target = nums[i] # 当计数器为 0 时，假设 nums[i] 就是众数
                count = 1
            elif nums[i] == target:
                count += 1 # 如果是target的话，类似正电子
            else:
                count -= 1 # 反之带负电
        return target # 因为存在众数，count必然>0，此时target就是众数

for line in sys.stdin:
    n = int(line.strip())
    nums = list(map(int, sys.stdin.readline().split()))
    result = Solution().majorityElement(nums)
    print(result)
```


# Complexity Analysis
- 时间复杂度：O(n)
- 空间复杂度：O(1)，如果用哈希表dict的话空间复杂度是O(n)
