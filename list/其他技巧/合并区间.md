# Problem Description
给你一个区间数组 intervals，其中 intervals[i] = [start_i, end_i]，请你合并所有重叠的区间，并返回一个不重叠的区间数组，该数组需恰好覆盖输入中的所有区间。

输出顺序约束：返回的区间必须按起点（start）升序排列，以保证答案唯一。



# Solution
注意：返回的区间必须按起点（start）升序排列，以保证答案唯一，所以先对intervals中的小数组排序

语法：intervals.sort(...):调用列表的 sort 方法原地排序；key=...:告诉 sort 按什么"依据"来比较大小；lambda x: x[0]是匿名函数

然后从start最小的数组开始比，如果left和right的关系不对就修改小数组的right（left不用改变因为已经sort过）


# Code

## LC version

```python
class Solution:
    def merge(self, intervals: List[List[int]]) -> List[List[int]]:
        # 区间需要合并：一个数组的right<=另一个的left，合并为[left1,right2]
        # 类似滑动窗口，第一组合并完的结果继续和剩下的数组比对
        # 可以按照start重新排一下intervals

        ans = [] # 按照start顺序确认没问题
        intervals.sort(key = lambda x:x[0])
        ans.append(intervals[0])

        for i in range(1, len(intervals)):
            cur = intervals[i]
            last = ans[-1]
            if cur[0] <= last[-1]: # 一个的right<=另一个的left
                last[-1] = max(last[1],cur[1]) # 只要改变right因为left已经排过序了，肯定是那个小的
            else:
                ans.append(cur)

        return ans
```

## ACM version

**ACM 模式的注意点：**

- 需要 import 完整的类（包括 sys、typing 等）
- 数据在标准输入流 stdin 中，全部是原始的文本字符串
- 必须用 print() 手动将结果写到标准输出流 stdout
- 需要写 while 或 for line in sys.stdin 循环处理，直到文件结束（EOF）

```python
import sys
from typing import List

class Solution:
    def merge(self, intervals: List[List[int]]) -> List[List[int]]:
        # 区间需要合并：一个数组的right<=另一个的left，合并为[left1,right2]
        # 类似滑动窗口，第一组合并完的结果继续和剩下的数组比对
        # 可以按照start重新排一下intervals

        ans = [] # 按照start顺序确认没问题
        intervals.sort(key = lambda x:x[0])
        ans.append(intervals[0])

        for i in range(1, len(intervals)):
            cur = intervals[i]
            last = ans[-1]
            if cur[0] <= last[-1]: # 一个的right<=另一个的left
                last[-1] = max(last[1],cur[1]) # 只要改变right因为left已经排过序了，肯定是那个小的
            else:
                ans.append(cur)

        return ans

data = sys.stdin.read().split()
idx = 0
while idx < len(data):
    n = int(data[idx]); idx += 1
    intervals = []
    for _ in range(n):
        intervals.append([int(data[idx]), int(data[idx + 1])])
        idx += 2
    result = Solution().merge(intervals)
    for interval in result:
        print(f"{interval[0]} {interval[1]}")
```


# Complexity Analysis
- 时间复杂度：O(n)
- 空间复杂度：O(1)
