# Problem Description
给你一个满足下述两条属性的 m x n 整数矩阵：每行中的整数从左到右按非严格递增顺序排列。每行的第一个整数大于前一行的最后一个整数。给你一个整数 target ，如果 target 在矩阵中，返回 true ；否则，返回 false 。


# Key Points
可以把二维数组拉成一维的，用标准二分模板即可
难点：知道了一维的索引，如何对应到二维横纵坐标？


# Code

## LC version

```python
class Solution:
    def searchMatrix(self, matrix: List[List[int]], target: int) -> bool:
        # 可以把二维数组拉成一维的，用标准二分模板即可
        m = len(matrix) # 行3
        n = len(matrix[0]) # 列4
        left = 0
        right = m * n - 1
        while left <= right:
            mid = (left + right) // 2
            if self.get(matrix, mid) == target:
                return True
            elif self.get(matrix, mid) < target:
                left = mid + 1
            elif self.get(matrix, mid) > target:
                right = mid - 1
        return False

    # 难点：知道了一维的索引，如何对应到二维横纵坐标？
    def get(self, matrix, index):
        m = len(matrix) # 行3
        n = len(matrix[0]) # 列4
        # 二维矩阵的横坐标
        i = index // n
        # 二维矩阵的纵坐标
        j = index % n
        return matrix[i][j]
```

## ACM version

**ACM 模式的注意点：**

- 需要 import 完整的类（包括 sys、typing 等）
- 数据在标准输入流 stdin 中，全部是原始的文本字符串
- 必须用 print() 手动将结果写到标准输出流 stdout
- 需要写 while 或 for line in sys.stdin 循环处理，直到文件结束（EOF）

```python
import sys
from typing import List

class Solution:
    def searchMatrix(self, matrix: List[List[int]], target: int) -> bool:
        # 可以把二维数组拉成一维的，用标准二分模板即可
        m = len(matrix) # 行3
        n = len(matrix[0]) # 列4
        left = 0
        right = m * n - 1
        while left <= right:
            mid = (left + right) // 2
            if self.get(matrix, mid) == target:
                return True
            elif self.get(matrix, mid) < target:
                left = mid + 1
            elif self.get(matrix, mid) > target:
                right = mid - 1
        return False

    # 难点：知道了一维的索引，如何对应到二维横纵坐标？
    def get(self, matrix, index):
        m = len(matrix) # 行3
        n = len(matrix[0]) # 列4
        # 二维矩阵的横坐标
        i = index // n
        # 二维矩阵的纵坐标
        j = index % n
        return matrix[i][j]


data = sys.stdin.read().split()
index = 0
while index < len(data):
    m = int(data[index]) # 一共几行
    index += 1
    n = int(data[index]) # 每行几个
    index += 1
    target = int(data[index])
    index += 1
    
    matrix = []
    for i in range(m):
        row = [int(x) for x in data[index:index + n]] # 每一行
        matrix.append(row)
        index += n

    result = Solution().searchMatrix(matrix, target)
    
    if result == True:
        print('true')
    else:
        print('false')
```
