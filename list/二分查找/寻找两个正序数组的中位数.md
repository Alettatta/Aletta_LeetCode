# Problem
https://labuladong.online/zh/problem/leetcode/median-of-two-sorted-arrays/description/


# Problem Description
给定两个大小分别为 m 和 n 的正序（从小到大）数组 nums1 和 nums2，请你找出并返回这两个正序数组的中位数。

输出精度要求：

为保证答案唯一，输出的中位数需保留 5 位小数（四舍五入）。


# Key Points
分别遍历两个数组：两个指针 i, j 分别从 nums1 和 nums2 的开头出发，谁指向的元素更小就让谁前进

但是这道题不需要求出整个合并数组，只要关心中间位置的元素：

设总长度L=m+n，中位数就在合并序列的下标 ⌊L/2⌋ 处（L为偶数时还需要前一个位置的值来取平均）


# Code

## LC version

```python
from typing import List


class Solution:
    def findMedianSortedArrays(self, nums1: List[int], nums2: List[int]) -> float:
        m, n = len(nums1), len(nums2)
        L = m + n # 如果合并数组的总长度
        half = L // 2 # 取得是下整:3-1；4-2

        i = j = 0
        prev = curr = 0
        for _ in range(half + 1): # half+1保证可以遍历到中位数（上面取下整的后一位）
            prev = curr
            # 双指针归并：哪个元素小就前进
            if i < m and (j >= n or nums1[i] <= nums2[j]): 
                # 注意j>=n这个条件，越界的时候也是加上i这个元素；同时j<=n要放在前面不然会越界
                curr = nums1[i]
                i += 1
            else:
                curr = nums2[j]
                j += 1

        if L % 2 == 0:
            return (prev + curr) / 2.0
            
        return float(curr)
```

## ACM version

**ACM 模式的注意点：**

- 需要 import 完整的类（包括 sys、typing 等）
- 数据在标准输入流 stdin 中，全部是原始的文本字符串
- 必须用 print() 手动将结果写到标准输出流 stdout
- 需要写 while 或 for line in sys.stdin 循环处理，直到文件结束（EOF）

```python
import sys
from typing import List


class Solution:
    def findMedianSortedArrays(self, nums1: List[int], nums2: List[int]) -> float:
        m, n = len(nums1), len(nums2)
        L = m + n # 如果合并数组的总长度
        half = L // 2 # 取得是下整:3-1；4-2

        i = j = 0
        prev = curr = 0
        for _ in range(half + 1): # half+1保证可以遍历到中位数（上面取下整的后一位）
            prev = curr
            # 双指针归并：哪个元素小就前进
            if i < m and (j >= n or nums1[i] <= nums2[j]): 
                # 注意j>=n这个条件，越界的时候也是加上i这个元素；同时j<=n要放在前面不然会越界
                curr = nums1[i]
                i += 1
            else:
                curr = nums2[j]
                j += 1

        if L % 2 == 0:
            return (prev + curr) / 2.0
            
        return float(curr)



for line in sys.stdin:
    header = line.strip()
    if not header:
        continue
    parts = header.split()
    if len(parts) < 2:
        break
    m = int(parts[0])
    n = int(parts[1])

    line1 = sys.stdin.readline()
    line2 = sys.stdin.readline()

    nums1 = list(map(int, line1.split())) if m > 0 else []
    nums2 = list(map(int, line2.split())) if n > 0 else []

    result = Solution().findMedianSortedArrays(nums1, nums2)

    print(f"{result:.5f}")
```


# Complexity Analysis
- 时间复杂度 O(m+n)，只遍历了合并序列的前一半。
- 空间复杂度 O(1)，只用了常数个变量。
